PNP and ADMM for image denoising and compressive sensing

ADMM or Alternating direction method of multiplier(ADMM) is a widely used algorithm for solving constrained optmization problems in image image restoration. In this project we use BM3d as a denoising algorthm to help ADMM to converge at fixed point under a continuation scheme. This increase the accuracy of denoising and deblurring. Also in the same project I have implemented ADMM for image restoration. For the code got to the Github link

Homography estimation and image stitching

In this project we created various classical and deep learning model(Homography NET & unsupervised deep Homography model) to calculate the homography of 2 frames and then further wrap the images to get a panoroma out of it.For the code got to the Github link

Deblur due to camera motion

Devised a multi scale Xception style UNET model with different loss function (L2 loss, L1 loss, PSNR )and GAN model to deblurs images using tensorflow. Report link

Depth through stereo CAM
